curl 命令详解及使用

发表于 2019-09-21

前言

简介

curl是一种命令行工具，用于传输使用URL语法指定的数据.是一个非常强大的工具,支持多种协议.详情看官网介绍
官网地址：https://curl.haxx.se/
GitHub地址：https://github.com/curl/curl

curl 帮助文档

https://www.cnblogs.com/shizhijie/p/8297840.html

curl 使用

  • curl -I www.baidu.com 显示响应首部信息
  • curl -s www.baidu.com 静默模式,不显示curl进度
  • curl -v www.baidu.com 显示一次通信详情信息
  • curl -v –trace-time www.baicu.com 显示详情并时间戳跟踪
  • curl -X GET www.baidu.com 指定 GET 请求访问百度,同 curl -G www.baidu.com
  • curl -s -X POST https://fanyi.baidu.com/v2transapi -d “from=zh&to=en&query=%E6%B5%8B%E8%AF%95&transtype=translang” 指定 POST 请求调用百度翻译将 “测试” 翻译为英文
  • curl -A “Chrome/70.0.3538.25” www.baidu.com 伪装chrome客户端访问百度
  • curl -H “name:zzw” www.baidu.com 自定义Header至百度
